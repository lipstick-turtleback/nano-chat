# LexiChat

> **Your AI growth companion.** Eleven unique mentors. Infinite potential.

LexiChat is a beautiful, locally-run personal development app with AI companions — each specializing in a different area of self-improvement. Powered by a local Express + SQLite server, Ollama AI models, and a sleek React frontend.

## 🚀 Quick Start

### Prerequisites

- **Node.js 18+**
- **Ollama** running locally (for AI provider)

### Install & Run

```bash
npm install
npm run dev
```

This starts both the Express server (port 3001) and Vite dev server (port 5173) concurrently.

### Using Ollama

```bash
# Install Ollama: https://ollama.com
ollama pull gemma4:31b-cloud
ollama serve
```

Then select Ollama as your provider in the Settings panel (⚙️ icon in header).

## 🏗 Architecture

```
Browser (Vite, port 5173)
    ↓ fetch('/api/*')
Express Server (port 3001)
    ↓
├── Ollama Proxy → localhost:11434 (configurable via .env)
├── Game Engine → Dice rolling, DnD scene generation
├── Challenge Generator → LLM-powered creative quizzes
├── Knowledge Extractor → Background analysis of chat logs
├── SQLite → player data, sessions, knowledge, campaigns
└── File Cache → Generated content caching
```

### Key Directories

```
src/                          # React frontend
├── App.jsx                   # Root component
├── styles/app.scss           # All styles (SCSS)
└── lib/
    ├── ai/                   # AI client interfaces
    ├── state/useStore.js     # Zustand store (all state + actions)
    ├── components/           # React components
    ├── services/             # Knowledge service, player stats
    └── utils/                # Constants, markdown, dice roller, themes

server/                       # Express backend
├── index.js                  # Server entry point
├── db.js                     # SQLite setup + schema
├── .env                      # Configuration
├── routes/                   # API route handlers
├── services/                 # Ollama, knowledge extraction
└── __tests__/                # Server route tests
```

## ⚙️ Configuration

### Environment Variables (`.env`)

```env
PORT=3001
NODE_ENV=development
OLLAMA_URL=http://localhost:11434
OLLAMA_DEFAULT_MODEL=gemma4:31b-cloud
DB_PATH=./server/data/lexichat.db
KNOWLEDGE_EXTRACT_INTERVAL_MESSAGES=10
```

### Settings Panel

Accessible via ⚙️ icon in the header:
- **Font Size**: Small (14px) → Extra Large (20px)
- **AI Provider**: Chrome AI or Ollama
- **Ollama Model**: Dropdown with refresh button
- **Data**: Export chat, clear chat
